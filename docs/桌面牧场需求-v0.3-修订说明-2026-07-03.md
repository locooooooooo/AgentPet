## 一、对 `桌面牧场需求-v0.3.md` 的具体条款修订

### 1.1 §〇·quarter §3.3（同源约束）修订

**原文**：
> 桌面牧场不能"独立"成为另一个产品 —— 它必须依附卖点

**修订后**：
> 桌面牧场不能"独立"成为另一个产品 —— 它必须依附卖点。
>
> **同源分级**：
> - **强同源（必须逐字符一致）**：`var(--agent-accent)` 染色、`STATE_METAS.expression` 表情脚本、accent ring / box-shadow 雾化配方
> - **弱同源（必须等价映射）**：keyframes 曲线（允许 ranch- 前缀，数值必须与 index.css 同名 keyframes 对齐）
> - **可差异化（允许脱离）**：emoji 字符本体（Q2 已默认"摆脱 4 牛 4 马"，8 个不同物种），用于强化"差异化身份"卖点
>
> **维护契约**：每次 `src/index.css` 调整 shake/breathe/blink 曲线，**Mavis 必须自动 diff `src/ranch/styles/ranch.css` 等价性**（脚本化检测，纳入 `orchestration:check`）。

### 1.2 §〇·quinary §3（3 级信息层级）修订

**原文**：
> ❌ 第 4 级层级（L0"全空"、L4"详细面板"）

**修订后**：
> ❌ 第 4 级层级（L0"全空"、L4"详细面板"）—— **例外**：临时 L4 选中态（点击动物后 ≤ 1.5s 内显示 agent 名 + codename + 当前 quote；超时淡出；不永久驻留）
>
> 临时 L4 不计入视觉层级（按"瞬时信息"处理），但**必须满足 1.5s 自动淡出 + 不堆叠多条**两个约束。

### 1.3 §〇·quinary §4（围栏"嵌入"视觉）修订

**原文**：
> - ❌ 不画草地 / 远景 / 服务器机柜等"游戏场景"
> - 内部背景：**完全透明**

**修订后**：
> - ❌ **禁止**：彩色草地色块 / 远山 / 服务器机柜 / 像素风贴图 / 进度条
> - ✅ **允许**（不破坏"嵌入感"前提下）：
>   - 极弱雾化（不透明度 ≤ 5%，仅用于桌面透明背景下的视觉锚定，不形成"色块"）
>   - 1px 网格 hairline（`rgba(255,255,255,0.04)` 左右，仅暗示分区，**不形成"地板"**）
>   - 工位区域的"软"暗示（`< 8px` 直径的发光圆点，agent accent 染色）
> - 圈外 / 围栏外：**完全透明**（desktop mode + 圈外穿透是 FR-008.1）
> - 围栏内允许 ≤ 5% 不透明度的弱装饰，**但不能形成可识别的"场景"**（用户余光扫到不能识别为"草地 / 网格地板 / 桌面贴图"）

### 1.4 §〇·quinary §5（L3 入口）修订

**原文**：
> | **L3 入口** | "召唤控制舱"图标 + "设置"图标 + "切模式"图标 | 24px 极小图标 | 鼠标 hover 牧场时显形 | 5% |

**修订后**：
> | **L3 入口** | 右键系统菜单（主触发，**v0.3 实施**）| 系统菜单原生样式 | 鼠标在牧场内任意位置右键 | 5% |
> | **L3' 后续** | 3 个 24px 图标 hover 显形 | 24px 极小图标（lucide-react）| 鼠标 hover 牧场时显形 | 5% |
>
> **v0.3 实施**：L3 = 右键系统菜单（`ranch:right-click` IPC + `Menu.buildFromTemplate`，见工程需求 §5.2 + FR-005）
> **v0.4 后续**：L3' = 3 个 hover 图标（召唤控制舱 + 设置 + 切模式），与右键菜单并存（hover 图标为主触发，右键菜单保留为高级入口）
> **不冲突**：右键菜单与 hover 图标可同时存在，不互相替换。

### 1.5 §〇·quinary §6（围栏视觉）修订

**原文**：
> | 围栏边框 | 1px hairline，**`var(--border)` 色（沿用既有）**，**不要** accent 色描边 |

**修订后**：
> | 围栏边框 | `2~8px double` 或 `1~2px solid`，**`var(--border)` 主体色**（沿用既有），可叠加 accent 色描边作为"装饰层"（不替换主边框色）|
> | 围栏 4 角 | 4 个 6~10px accent 色圆点，**hover 时显形**（默认隐形）|
> | 围栏阴影 | 极弱 0 1px 3px 内阴影（让圈"嵌入"桌面），外阴影 ≤ 8px 视觉过渡 |

### 1.6 §〇·bis §2（动画曲线）修订

**原文**：
> | 动画曲线 | `shake 520ms ease-in-out infinite` / `breathe 3.6s ease-in-out infinite` / `blink 4s ease-in-out infinite` | 动物动效（HUD 通知）|

**修订后**：
> | 动画曲线 | **等价映射**：`shake/breathe/blink` 在 `src/index.css` 与 `src/ranch/styles/ranch.css` 必须**数值对齐**（duration / easing / iteration count 一致），允许 `ranch-` 前缀作为 namespace 区分 |
> | **维护契约** | `scripts/check-orchestration.mjs` 增加 keyframes 等价性 diff 检测：解析 `index.css` 与 `ranch.css` 的同名 keyframes，duration 偏差 > 5% 报错 |
> | **当前状态** | `ranch.css` 9 个 keyframes 与 `index.css` 数值已对齐（shake 520ms / breathe 3.6s / blink 4s），曲线幅度允许微调（如 shake ±2px vs ±1px）|

## 六、orchestration:check 新增检测（v0.3 M5 验收用）

```javascript
// scripts/check-orchestration.mjs 新增（v0.3 M5 验收）
function checkRanchVisualGuard() {
  const ranchCss = readFileSync('src/ranch/styles/ranch.css', 'utf-8');
  const indexCss = readFileSync('src/index.css', 'utf-8');
  
  // 1. 围栏边框 1~8px
  const fenceBorderMatch = ranchCss.match(/border:\s*(\d+)px\s+(double|solid)/);
  assert(fenceBorderMatch && [+fenceBorderMatch[1]] >= 1 && [+fenceBorderMatch[1]] <= 8,
    '围栏边框必须 1~8px');
  
  // 2. 4 角圆点 6~10px
  const cornerDotMatch = ranchCss.match(/border-radius:\s*(\d+)px/);
  assert(cornerDotMatch && [+cornerDotMatch[1]] >= 6 && [+cornerDotMatch[1]] <= 10,
    '围栏 4 角圆点必须 6~10px');
  
  // 3. 背景不透明度 ≤ 5%
  const bgOpacityMatches = [...ranchCss.matchAll(/rgba\(\s*\d+\s*,\s*\d+\s*,\s*\d+\s*,\s*([\d.]+)\s*\)/g)];
  const maxBgOpacity = Math.max(...bgOpacityMatches.map(m => +m[1]));
  assert(maxBgOpacity <= 0.05,
    `背景不透明度 ≤ 5%，当前最大值 ${maxBgOpacity}`);
  
  // 4. keyframes duration 等价性
  const indexKeyframes = parseKeyframes(indexCss);
  const ranchKeyframes = parseKeyframes(ranchCss);
  for (const [name, duration] of Object.entries(indexKeyframes)) {
    if (ranchKeyframes[name]) {
      const diff = Math.abs(ranchKeyframes[name] - duration) / duration;
      assert(diff <= 0.05,
        `keyframes ${name} duration 偏差 > 5%（index=${duration}ms, ranch=${ranchKeyframes[name]}ms）`);
    }
  }
  
  // 5. SelectedOverlay 1.5s 自动淡出
  const overlayCss = ranchCss.match(/\.selected-overlay\s*\{[^}]*\}/);
  assert(overlayCss && overlayCss[0].includes('animation') && overlayCss[0].match(/(\d+)ms/)?.[1] <= 1500,
    'SelectedOverlay 必须 ≤ 1.5s 自动淡出');
  
  return { pass: true, message: 'RANCH_VISUAL_GUARD 全过' };
}
```
