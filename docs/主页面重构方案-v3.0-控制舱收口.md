## 二、v3.0 实施分阶（P0 / P1 / P2）

### P0 阶段（2026-07-03 起，**最高优先级**）

> **目标**：解决控制舱 4 个根因中的 ① ② ③（信息密度问题），同时建立"页面安静度"基线。
> **预计工时**：8~12 小时
> **不动**：中央 8 卡 / 既有色板 / 既有动画 / 既有状态文案

#### P0-1 orchestration 区视觉权重分级（2h）

**问题**：5 张 task card 同权重，用户不知道看哪张

**方案**：
- 当前选中（activeAgentId 匹配）的 task card：`scale(1.05)` + accent 描边 + box-shadow 雾化
- 其他 4 张：`opacity: 0.55` + 灰显
- 鼠标 hover 任一 card → 该 card 临时高亮 + 其他 card 进一步降低到 `opacity: 0.4`

**修改文件**：
- `src/components/NiuMaWorkspace.tsx` orchestration 区
- `src/index.css` 新增 `.task-card-active` / `.task-card-dimmed` 类

**验收**：
- 控制舱 5 张 task card 中 1 张高亮 4 张灰显
- hover 任一 card → 该 card 临时高亮
- 卖点文件（`NiuMaAvatar.tsx` / 中央 4×2 grid / `is-hot` keyframe）**0 字节变动**

#### P0-2 connectors + status 合并为 StatusStrip（2h）

**问题**：6 chip + 3 connector card 两排并列，视觉无主次

**方案**：
- 单行 StatusStrip（高度 36px）：左 `connector · ready` · 中 `tasks: 12/30` · 右 `last event: 3s 前`
- 折叠为顶部一行（不占垂直空间）
- 鼠标 hover → 下拉显示完整 connector card 列表

**修改文件**：
- 新增 `src/components/StatusStrip.tsx`
- `src/index.css` 新增 `.status-strip` / `.status-strip-dropdown` 类
- `src/components/NiuMaWorkspace.tsx` 替换原 connectors + status 区域

**验收**：
- StatusStrip 单行显示 3 个核心指标
- hover 显示完整 connector 列表
- 不破坏既有 chip 颜色 / 状态语义

#### P0-3 右侧详情面板 Tab 化（3h）

**问题**：右侧 30% 区域 8 块信息堆叠，"一堵墙"

**方案**：
- 右侧面板改为 Tab 容器：`下发任务` / `任务队列` / `流式日志` 三 Tab
- 默认 Tab = `下发任务`（最高频）
- Tab 切换 = 横向动画（slide，180ms）
- 不动既有文案 / 状态码 / 按钮位置

**修改文件**：
- `src/components/PortraitWorkspacePanel.tsx`（或当前右侧面板组件）
- 新增 Tab 状态管理（useState 或 Context）

**验收**：
- 默认显示"下发任务"Tab
- 3 个 Tab 可切换
- Tab 切换动画 ≤ 200ms
- 不破坏既有"下发任务"表单 / "任务队列"列表 / "流式日志"组件

#### P0-4 删除中央"批发排障"浮窗（或挪到右下角小三角）（1h）

**问题**：中央区 4 卡 + 1 浮窗形成视觉干扰

**方案**（二选一，由用户拍板）：
- **A 删**：彻底删除该浮窗（最克制）
- **B 挪**：挪到右下角小三角按钮，hover 展开 200×200 浮层

**修改文件**：
- `src/components/NiuMaWorkspace.tsx` 中央浮窗

**验收**：
- 中央 4 卡视觉无干扰
- 浮窗功能不丢失（仅位置/形态变化）

## 三、v3.0 视觉规范（控制舱新 UI 的硬约束）

### 3.1 视觉层级基线

| 层级 | 内容 | 视觉权重 |
|---|---|---|
| L0 | 背景 / 网格 | 不透明度 ≤ 5%（与桌面牧场 §〇·quinary §4 一致）|
| L1 | 中央 8 卡 + 顶 StatusStrip + 右侧 Tab 容器 | 100% |
| L2 | 当前选中 task card / 选中 8 卡 | 105% scale + accent 描边 |
| L3 | 卡片 hover 浮起 / Tab 内容 | opacity transition |
| L4 | 操作按钮（hover 露）| 半透明悬浮条 |
| L5 | 调试信息 / metric（hover 露）| 极小字 |

**控制舱允许 5 级**（v0.3 §〇·quinary §末允许），但 L4/L5 必须 hover 露（不能常驻）。

### 3.2 配色 / 动画 / emoji 沿用

| 类目 | 沿用源 |
|---|---|
| 色板 | `--green / --blue / --orange / --red / --panel / --border` |
| accent | `var(--agent-accent)` |
| 表情 | `STATE_METAS.expression` |
| emoji | `agent.avatar` |
| 动画 | 沿用既有 `shake / breathe / blink`（不引入新曲线）|

### 3.3 控制舱文案黑话化（与桌面牧场对齐）
