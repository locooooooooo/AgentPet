# 🐄 多Agent牛马看板主页面重构方案

> **⚠️ 本方案已被 [`docs/主页面重构方案-v3.0-控制舱收口.md`](./主页面重构方案-v3.0-控制舱收口.md) 收口，本文档仅供历史参考。**
> **收口日期**：2026-07-03
> **收口原因**：与"视觉降噪 / 极简 v2.0"方向冲突（重装饰 vs 重克制），统一归 P0/P1/P2 分阶实施。v1.0 的"数据管道流光 / 拖拽派活 / 抛物线投喂"等内容挪 v0.4+。

本方案旨在重构多Agent看板（Cockpit Workspace）的主页面布局，紧扣**“幽默牛马 Emoji 动画”**这一核心卖点，针对**“多Agent状态打通”**这一技术壁垒进行可视化设计，并全面考虑响应式适配与现代 UI/UX 审美。

---

## 一、 视觉风格与 UI/UX 审美定位

### 1. 视觉风格：赛博格子间 (Cyber-Cubicle) + 玻璃拟态 (Glassmorphism)
*   **底色与光晕**：采用深邃的太空蓝黑（`#020617`）作为背景，搭配高饱和度的霓虹状态色（摸鱼绿 `#22c55e`、加班金 `#f59e0b`、流汗蓝 `#38bdf8`、警报红 `#f87171`）。
*   **卡片质感**：卡片采用半透明玻璃质感（`backdrop-filter: blur(12px)`），搭配极细的彩色发光边框（Glow Border），边框颜色与当前 Agent 的状态实时绑定。

### 2. 文案包装：全面“牛马黑话”化
*   “运行任务” $\rightarrow$ **“正在拉磨”**
*   “已归档任务” $\rightarrow$ **“历史功绩”**
*   “体力能量值” $\rightarrow$ **“牛马剩余电量”**
*   “CPU 温度” $\rightarrow$ **“脑门发热度”**
*   “下发新任务” $\rightarrow$ **“画饼/派活”**

---

## 二、 核心卖点：牛马 Emoji 动画与状态打通可视化

为了凸显“打通每个 Agent 状态”这一技术壁垒，在 UI 上进行强视觉反馈：

### 1. “数据管道”流光动画 (Data Pipeline Glow)
*   **设计**：在左侧的 **Connector 管控区** 与中央的 **牛马工位** 之间，绘制一条半透明的“数据管道线”。
*   **动态**：当底层 Connector 成功抓取到 Agent 的真实进程状态时，管道线上会有流光（CSS Gradient Animation）从左侧流向对应的 Agent 卡片，直观展示“状态已打通”。

### 2. 场景化“格子间”动画 (Cubicle Stage)
将原本单一的圆形头像框，重构为**“工位格子间”**舞台：
*   **摸鱼态 (Slacking)**：Emoji 左右惬意晃动，背景出现微缩的“椰子树与沙滩”粒子，气泡台词：“在写了在写了（指新建文件夹）”。
*   **加班态 (Overtime)**：Emoji 剧烈抖动，四周有 💀 粒子向上飘散，背景变为暗红色火焰，气泡台词：“再改我是狗”。
*   **崩溃态 (Panicking)**：Emoji 上下剧烈跳动，格子间四周闪烁红色警报光晕（🚨），气泡台词：“生产环境炸了！！”。
*   **发版态 (Deploying)**：Emoji 3D 旋转，四周有 🚀 粒子，气泡台词：“向生产环境祈祷中...”。

## 三、 响应式布局重构方案 (Responsive Grid)

针对不同分辨率，重构目前的 `cockpit-grid` 三栏布局，解决低分辨率下右侧面板折行导致的视觉混乱：

```
┌────────────────────────────────────────────────────────────────────────┐
│                             Header (全局 KPI)                          │
├───────────────────────┬────────────────────────┬───────────────────────┤
│                       │                        │                       │
│   Left Rail (280px)   │   Center Stage (1fr)   │   Right Rail (400px)  │
│   [ 极简调度与网关 ]  │   [ 牛马工位矩阵 ]     │   [ 选中牛马控制台 ]  │
│                       │                        │                       │
└───────────────────────┴────────────────────────┴───────────────────────┘
```

### 1. 超宽屏/大屏设备 ($\ge 1440px$)
*   **布局**：保留三栏布局，但优化比例（`280px minmax(0, 1fr) 400px`）。
*   **优化**：左轨收缩，将空间让给中央的**牛马工位矩阵**，确保 8 个 Agent 的 Emoji 动画有足够的展示空间。

### 2. 中等屏幕 ($1024px - 1439px$)
*   **布局**：自动转为**双栏布局**。
*   **重构点**：
    *   左侧“调度总览”与“Connector管控”折叠为顶部通栏或左侧极简 Icon 悬浮条。
    *   右侧“操作面板”改为**“抽屉式 (Drawer)”**。默认收起，点击任意 Agent 卡片时，从右侧平滑滑出，不占用常驻空间，把 100% 的宽度留给牛马矩阵。

### 3. 小屏/移动端 ($< 1024px$)
*   **布局**：**单栏流式布局**。
*   **重构点**：
    *   牛马矩阵自动适配为 2 列或 1 列网格。
    *   点击 Agent 卡片后，详情与终端日志通过**底部抽屉 (Bottom Sheet)** 向上弹出，提供沉浸式操作体验。

## 四、 趣味交互动画重构 (Micro-interactions)

增加动作按钮与 Agent 头像之间的联动动画（使用 `framer-motion`）：

1.  **画饼 (Pie) 投喂**：点击“画饼加班”按钮，一个 🥧 Emoji 从鼠标处飞入 Agent 格子间，Agent 瞬间切换为 `overtime` 状态，并伴随火焰粒子。
2.  **咖啡 (Coffee) 续命**：点击“给杯咖啡”，☕ 飞入格子间，能量条瞬间回满，并伴随绿色光环扩散。
3.  **鞭策 (Whip) 强制开工**：点击“强制开工”，⚡ 闪电劈中格子间，Emoji 产生剧烈震动。
4.  **鼠标悬停 (Hover/Poke)**：鼠标悬停在 Emoji 上时，触发“被戳”的受惊缩放动画，并随机更换一句幽默吐槽气泡。
