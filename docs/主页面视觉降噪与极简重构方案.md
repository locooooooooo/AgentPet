# 🐄 主页面视觉降噪与极简重构方案 (De-cluttering & Focus)

> **⚠️ 本方案已被 [`docs/主页面重构方案-v3.0-控制舱收口.md`](./主页面重构方案-v3.0-控制舱收口.md) 收口，本文档仅供历史参考。**
> **收口日期**：2026-07-03
> **收口原因**：本方案方向（删除左轨 + 抽屉化 + hover 露操作）与"极简 v2.0"方向一致，被合并进 v3.0 的 P0/P1 阶段执行。

针对当前界面“信息密度过载、重点不突出、开发调试信息过多”的问题，本方案提出**“无情降噪，回归牛马”**的极简重构方案。

---

## 一、 视觉降噪：删减非核心元素 (What to Remove)

### 1. 彻底移除左侧轨道 (Left Rail)
*   **问题**：左侧的“调度总览”、“角色分工与监督”、“状态管道管控”充斥着大量的开发调试文本，对于日常看板管理来说是严重的视觉噪音。
*   **重构**：**100% 隐藏左侧轨道**。将这些系统级调度数据收纳进顶部 Header 的一个极简“系统状态”图标中，点击时以轻量弹窗（Popover）展示，平时不占用任何屏幕空间。

### 2. 移除中央顶部的“主工作区” (Primary Worksurface)
*   **问题**：中央顶部的 `Codex · 1号舍` 面板与右侧的 `Operator Console` 以及下方的工位卡片高度重复，同一个 Agent 的信息在屏幕上出现了三次。
*   **重构**：**彻底删除该面板**。将空间完全释放给下方的**牛马工位矩阵**，让牛马们的 Emoji 动画成为视觉的绝对中心。

### 3. 清理 Header 调试文案
*   **问题**：Header 下方的介绍文案（“深色高对比控制台，左轨追踪调度真源...”）属于开发备注，不应出现在正式 UI 中。
*   **重构**：**彻底删除该行文案**，使 Header 高度从 120px 缩减至 60px，大幅释放垂直空间。

---

## 二、 界面重组：三栏变单栏/双栏 (Layout Restructuring)

清理后的界面将变得极其干净、聚焦：

```
┌────────────────────────────────────────────────────────────────────────┐
│                      Header (仅保留 Logo、极简 KPI)                    │
├──────────────────────────────────────────────┬─────────────────────────┤
│                                              │                         │
│                                              │   Right Console         │
│             Center Stage (1fr)               │   (360px)               │
│             [ 干净的牛马工位矩阵 ]           │   [ 仅在选中牛马时 ]    │
│                                              │   [ 侧边滑出，平时隐藏 ]│
│                                              │                         │
└──────────────────────────────────────────────┴─────────────────────────┘
```

### 1. 聚焦“牛马工位矩阵” (Center Stage)
*   **卡片降噪**：
    *   默认隐藏卡片底部的 5 个操作按钮（画饼、咖啡、鞭策、摸鱼、派活）。**仅在鼠标悬停（Hover）在卡片上时，这些按钮才以半透明悬浮条的形式优雅浮现**。
    *   简化卡片文本：只保留“名称”和“状态标签”，移除“Codename（如批发排障）”和“Slot（如[1号舍]）”，这些信息移至右侧详情。
    *   合并进度条：将“电量”和“发热度”合并为一条极细的双色指示线，或者仅在悬停时显示。

### 2. 右侧控制台“抽屉化” (Drawer Console)
*   **重构**：右侧控制台默认收起。只有当用户点击矩阵中的某头牛马时，控制台才从右侧平滑滑出。
*   **控制台内部降噪**：
    *   去除控制台顶部的重复头像和描述。
    *   将“下发任务”、“任务队列”和“流式日志”折叠为**标签页 (Tabs)** 切换，避免在一个纵向面板里塞满所有内容。
